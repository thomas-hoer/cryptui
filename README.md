# cryptui

CryptUI is a file hosting system that provides a real end to end encryption. No personal data is shared with the server.
Every byte is getting encrypted on the client before 
by yet being fast and intuitive
CryptUI is an aproach to provide a secure file hosting provider. It features an easy to use client side encryption of all uploaded files and most of the meta informations. 

Visit preview on https://crpytui.de

**Warning:** The API is not stable yet. Data created with an older version might not fully work with newer versions. There won't be any update tools yet.

### Features
- Upload / Download with client side encryption
- Thumbnails and preview
- Directories
- Multi-Device

### Open Tasks / Planned Features
- Share Files with other users
- PWA
- Avoid data loss when using 2 devices at the same time
